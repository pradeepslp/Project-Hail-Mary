# 🚀 PROJECT HAIL MARY

### Autonomous Space Mission Intelligence Platform

> **"What if a spacecraft could think, reason, learn, collaborate, and survive on its own?"**

Project HAIL MARY is an advanced autonomous mission simulation platform that combines Digital Twins, Multi-Agent Systems, AI Decision Intelligence, Real-Time Physics Simulation, Mission Planning, Trajectory Analysis, and Conversational AI into a single interactive mission control environment.

Unlike traditional space simulators, HAIL MARY focuses on **autonomous decision-making under uncertainty**, allowing users to inject anomalies and observe how specialized AI agents collaborate to protect mission success.

---

# 🌌 Why This Project Matters

Future deep-space missions cannot rely on constant human supervision.

Communication delays, system failures, fuel constraints, and environmental hazards require spacecraft to make intelligent decisions autonomously.

Project HAIL MARY explores this future by simulating:

* Autonomous Mission Control
* Multi-Agent Collaboration
* AI-Powered Decision Making
* Spacecraft Digital Twins
* Mission Planning & Trajectory Analysis
* Real-Time Risk Assessment
* Autonomous Recovery Strategies
* Human-AI Interaction

# 🚀 Getting Started

## 1. Clone Repository

```bash
git clone <repository-url>
cd project-hail-mary
```

---

## 2. Backend Setup

```bash
cd backend

python -m venv venv

# Windows
venv\Scripts\activate

# Linux / Mac
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## 3. Configure Environment

Create:

```bash
backend/.env
```

Example:

```env
DATABASE_URL=postgresql://user:password@localhost:5432/hailmary

REDIS_URL=redis://localhost:6379

OPENAI_API_KEY=your_key

GEMINI_API_KEY=your_key

ELEVENLABS_API_KEY=your_key
```

---

## 4. Start PostgreSQL

Ensure PostgreSQL is running.

Create database:

```sql
CREATE DATABASE hailmary;
```

---

## 5. Start Redis

```bash
redis-server
```

Verify:

```bash
redis-cli ping
```

Expected:

```text
PONG
```

---

## 6. Start Backend

```bash
cd backend

uvicorn main:app --reload
```

Backend:

```text
http://localhost:8000
```

---

## 7. Frontend Setup

```bash
cd frontend

npm install
```

Run:

```bash
npm run dev
```

Frontend:

```text
http://localhost:3000
```
